# lazyvim-config

## Installation

Make a backup of your current Neovim files:

```bash
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

Clone the repository:

```bash
git clone https://github.com/lelouvincx/lazyvim-config.git
```

Start neovim:

```bash
nvim
```
